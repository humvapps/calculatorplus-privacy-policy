Privacy Policy

Calculator+  ·  com.calcplus.vault  ·  Effective:

Summary

Calculator+ does not collect, transmit, or share any personal data. Everything stays on your device. There are no servers, no accounts, and no analytics.

This policy explains in detail what data the app stores locally, why it stores it, and how you can delete it at any time.

What Calculator+ Stores Locally

All data is stored only on your device, encrypted with AES-256-GCM. Nothing leaves your device unless you explicitly export a backup.

  • Vault file — an encrypted binary file containing your saved site entries, labels, counters, and template preferences. Stored in the app's private storage directory, inaccessible to other apps.
  • Settings — your preferences (lock timeout, clipboard clear duration, default template, autofill toggle). Stored in a plain JSON file in private app storage.
  • Biometric session key — if you enable biometric unlock, an encrypted session key is stored in Android Keystore / EncryptedSharedPreferences. This is cleared when you disable biometrics or change your master password.
  • Autofill CODE cache — if you opt in, a per-entry CODE value is stored in EncryptedSharedPreferences to speed up autofill. This cache is wiped entirely when you disable the Autofill feature.
  • Onboarding flag — a single boolean in SharedPreferences recording whether you have completed the first-launch walkthrough.

Your master password is never stored. It is used transiently in memory to derive encryption keys and is cleared from memory when the app locks.

What We Do Not Collect

  • No names, email addresses, or account information
  • No device identifiers (IMEI, advertising ID, Android ID)
  • No location data
  • No usage analytics or crash reporting
  • No passwords — only SHA-256 hashes (for verification) or AES-256 encrypted blobs (for manually saved passwords)
  • No data sent to any server — ever
Calculator+ has no internet permission. It cannot make network requests of any kind.

Android Autofill Service

If you enable the optional Autofill feature, Calculator+ registers as an Android Autofill provider. In this role it:

  • Reads the structure of the app or website requesting autofill (field types, domain name, package name) to match it to a saved entry. This information is processed entirely on-device and never stored or transmitted.
  • May observe the username field value when Android offers to save a login. This hint is stored locally in EncryptedSharedPreferences and only used to pre-fill the username field in future autofill sessions for the same site.
  • Requires biometric authentication every time it fills credentials into another app.
Disabling the Autofill toggle in Settings immediately wipes all stored autofill CODE values from the device.

Backups and Exports

Calculator+ offers two manual export options, both initiated explicitly by you:

  • Encrypted vault backup — exports the raw AES-256-GCM encrypted vault file. Without your master password this file is unreadable.
  • Password-protected JSON backup — exports vault entries encrypted with AES-256-CBC using a backup password you choose. Without that password the file is unreadable.

Android system backups (Google Backup) are disabled for this app via android:allowBackup="false". Your vault is never included in automatic device backups.

Third-Party Libraries

Calculator+ uses the following open-source libraries. None of them collect data or make network requests:

  • PointyCastle — cryptographic operations (AES, scrypt, HMAC-SHA256)
  • flutter_secure_storage — Android Keystore / EncryptedSharedPreferences wrapper
  • local_auth — biometric authentication
  • path_provider — device storage paths
  • file_picker — backup import file selection
  • share_plus — backup export via share sheet
  • shared_preferences — onboarding flag storage

Children's Privacy

Calculator+ is not directed at children under 13. We do not knowingly collect any information from children. If you believe a child has used this app, please contact us — though given that no data leaves the device, there is nothing to delete remotely.

Your Rights and Data Deletion

Because all data is stored only on your device, you have complete control at all times:

  • Delete all data — uninstall the app. Android will remove the vault file, settings, and all EncryptedSharedPreferences entries automatically.
  • Clear autofill cache — toggle Autofill off in Settings. All stored CODEs are wiped immediately.
  • Clear biometric session — disable Biometrics in Settings, or change your master password.

There is no account to delete, no server to contact, and no data held by anyone other than you.
